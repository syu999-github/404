# 404

this is a chrome extension。
屏蔽一些不良网站，打开后直接显示404。

## 原理

不良网站的url都放在一个黑名单中~~(blacklist.js)~~（blacklist.json 2018.08.04 update），当用户打开chrome标签页时，程序会遍历黑名单和浏览器地址栏里的地址做对比，如果能对上，则意味着该网站是不良网站，直接屏蔽掉。
